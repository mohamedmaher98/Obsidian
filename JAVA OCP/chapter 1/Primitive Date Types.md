# Primitive Data Types in Java

الPrimitive data types هي أنواع البيانات الأساسية في Java، مش كائنات (Objects)، ومستخدمة لتخزين القيم البسيطة.

## 1. Integer Types (أعداد صحيحة)

|النوع|الحجم|المدى|مثال|
|---|---|---|---|
|byte|1 بايت|-128 إلى 127|100|
|short|2 بايت|-32,768 إلى 32,767|30000|
|int|4 بايت|-2,147,483,648 إلى 2,147,483,647|200000|
|long|8 بايت|-9,223,372,036,854,775,808 إلى 9,223,372,036,854,775,807|5000000000|

## 2. Floating-point Types (أعداد عشرية)

|النوع|الحجم|دقة|مثال|
|---|---|---|---|
|float|4 بايت|~7 أرقام عشرية|3.14|
|double|8 بايت|~16 أرقام عشرية|3.141592|

## 3. Character Type (حرف واحد)

|النوع|الحجم|مثال|
|---|---|---|
|char|2 بايت|'A', 'م'|

## 4. Boolean Type (صح أو غلط)

|النوع|القيمة الممكنة|
|---|---|
|boolean|true / false|

## نصائح عملية

- للأعداد الطبيعية الصغيرة → استخدم `int`.
    
- للأعداد العشرية → استخدم `double`.
    
- للصح أو الغلط → استخدم `boolean`.
