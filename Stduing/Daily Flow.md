## التوزيع الأسبوعي للمواد

### الأسبوع 1-2: بناء الأساسيات

**الأيام الفردية (الأحد/الثلاثاء/الخميس)**: Java Fundamentals

- OOP Concepts
- Collections Framework
- Exception Handling

**الأيام الزوجية (الاثنين/الأربعاء)**: SQL Basics

- Basic Queries (SELECT, INSERT, UPDATE, DELETE)
- JOINs and Subqueries
- Database Design

### الأسبوع 3-4: المستوى المتوسط

**الأيام الفردية**: Advanced Java

- Multithreading
- Stream API
- Design Patterns

**الأيام الزوجية**: Advanced SQL

- Stored Procedures
- Views and Indexes
- Performance Optimization

### الأسبوع 5-6: التطبيق والممارسة

**الأيام الفردية**: Java Projects

- Spring Boot basics
- REST APIs
- Database Integration

**الأيام الزوجية**: Complex SQL

- Window Functions
- Advanced JOINs
- Real-world Scenarios

## نصائح للالتزام

### للحفاظ على الدافعية:

- سجل تقدمك يومياً
- احتفل بالإنجازات الصغيرة
- لا تكن قاسياً على نفسك في الأيام الصعبة

### للمرونة:

- إذا فاتتك جلسة صباحية، عوضها مساءً
- يمكن تبديل مواد اليوم حسب الظروف
- الجمعة والسبت للتعويض والمراجعة

### للتركيز:

- أغلق الهاتف أثناء الدراسة
- استخدم تقنية Pomodoro (25 دقيقة دراسة + 5 دقائق راحة)
- جهز المواد والبيئة مسبقاً
