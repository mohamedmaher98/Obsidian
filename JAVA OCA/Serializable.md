Serializable:
هو **Interface** في Java، موجود في `java.io.Serializable`.
وظيفته إنه يخلّي أي كائن (object) تقدر تحوّله لـ **stream من bytes** علشان:- 
تبعته عبر الشبكة (network)
- تحفظه في ملف
- تخزّنه في قاعدة بيانات
- أو تحتفظ بيه مؤقتًا في الكاش (مثل Radis)

الSerializable هو علامة (marker interface)، يعني مفيهوش أي دوال، بس وجوده في الكلاس بيقول للجافا:
"الكائنات اللي من النوع ده ممكن تتحوّل لبايتات (byte stream) بسهولة."

| الاستخدام                     | الشرح                                              |
| ----------------------------- | -------------------------------------------------- |
| **حفظ الكائنات في ملفات**     | زي لما تعمل save لإعدادات برنامج                   |
| **إرسال الكائنات عبر الشبكة** | لو بتعمل تطبيق عميل/خادم (Client/Server)           |
| **الكاش**                     | لما تستخدم مكتبات كاش زي Redis أو Ehcache          |
| **تخزين في DB**               | أحيانًا بتحوّل الكائن إلى JSON أو ByteArray وتخزنه |
| ال**Java RMI**                | لو بتستخدم Remote Method Invocation                |
