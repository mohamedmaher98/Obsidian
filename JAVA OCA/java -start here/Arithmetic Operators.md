الـ Arithmetic Operators هي الرموز اللي بنستخدمها علشان نعمل عمليات حسابية زي الجمع والطرح والضرب... إلخ.
## ✴️ أنواع الـ Operators:

|       |                |                |        |        |
| ----- | -------------- | -------------- | ------ | ------ |
| الرمز | الاسم          | الوظيفة        | مثال   | الناتج |
| +     | Addition       | جمع            | 5 + 3  | 8      |
| -     | Subtraction    | طرح            | 5 - 3  | 2      |
| *     | Multiplication | ضرب            | 5 * 3  | 15     |
| /     | Division       | قسمة (علي عدد) | 10 / 2 | 5      |
| %     | Modulus        | باقي القسمة    | 10 % 3 | 1      |
double x = 10;
double y = 3;
System.out.println(x / y); // 3.333...
## ⚠️ ملاحظات:

- القسمة بين `int` بتطلع `int` (بتهمل الكسر).
    
- القسمة بين `double` بتطلع نتيجة عشرية.
    
- `%` بيجيب **الباقي**، مفيد جدًا في التحقق من الأرقام الزوجية والفردية:
