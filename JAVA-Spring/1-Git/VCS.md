- Git بيخزن **snapshot لكل commit**.
    
- كل ملف بيتحول لـ **blob**، وGit بيستخدم hash للتأكد إنه ما يخزنش نسخة مكررة.
    
- المجلدات عبارة عن **tree**، وكل commit يشير للـ tree ده.
    
- الـ commits مرتبطة ببعض (parent) وده بيخلي عندك تاريخ كامل للمشروع.
