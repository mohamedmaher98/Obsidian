### 1️⃣ إيه هو `.gitignore`؟

الـ `.gitignore` ده ملف موجود جوه مشروع Git، وظيفته **تقول لـ Git إيه الملفات أو الفولدرات اللي متحبش يتحطوا في الـ repository**.

يعني مثلاً ملفات:

- فيها معلومات سرية (passwords، API keys)
    
- ملفات مؤقتة (logs، temp files)
    
- ملفات الـ build أو الـ IDE (زي `.class`, `target/`, `.idea/`, `node_modules/`)
    

لو Git شاف الملفات دي، هيهملها ومش هيضيفها أو يراقب تغييراتها.

### 2️⃣ شكل الملف

الملف عبارة عن **قائمة نصية**. كل سطر فيه **مسار أو نمط** لملفات/فولدرات.

مثال:
```bash

# تجاهل ملفات build
target/
build/

# تجاهل ملفات النظام المؤقتة
*.log
*.tmp

# تجاهل إعدادات IDE
.idea/
*.iml

# تجاهل node modules
node_modules/

```
- السطر اللي بيبدأ بـ `#` ده **تعليق**.
    
- `*` معناه أي حاجة (wildcard). مثال: `*.log` = كل الملفات اللي امتدادها `.log`.
    
- `/` في الآخر معناه **فولدر كامل**. مثال: `build/` = تجاهل فولدر build كله.
    
- `/` في الأول معناه **من الجذر**. مثال: `/secret.txt` = تجاهل الملف ده من الجذر بس.

### 4️⃣ حاجات مهمة لازم تعرفها

1. **مش هيشتغل على الملفات اللي اتضافت قبل كده**  
    لو الملف موجود في Git فعلاً، حط `.gitignore` مش هيشيله، لازم تمسحه من الـ repo الأول:
- **فيه `.gitignore` عام لكل مشروع**  
    ممكن تعمل `.gitignore` واحد على مستوى المشروع، أو على مستوى الـ user كله (زي ملفات الـ IDE).
    
- **ممكن تستخدم wildcards**
    
    - `*` = أي حاجة
        
    - `?` = أي حرف واحد
        
    - `[abc]` = أي حرف من a أو b أو c

### 1️⃣ إيه `git rm --cached`؟

-ال `git rm` = إزالة ملف من Git.
    
- ال`--cached` = إزالة الملف **من تتبع Git بس**، مش من جهازك. يعني الملف يفضل موجود عندك على الكمبيوتر لكن Git مش هيحطه تحت السيطرة بعد كده.
    

ده بالظبط اللي محتاجينه لو:

- أضفت ملف بالغلط قبل ما تحطه في `.gitignore`.
    
- وعايز Git يتجاهله من دلوقتي.

### 2️⃣ إزاي تستخدمه؟

مثال: عندك ملف `secret.txt` اتضاف بالخطأ:
git rm --cached secret.txt

دلوقتي:

- الملف اتشال من Git index.
    
- لسه موجود عندك على الكمبيوتر.
    
- لو حطيت `secret.txt` في `.gitignore`، Git مش هيحاول يضيفه تاني.

- ### 3️⃣ لو عايز تمسح فولدر كامل من Git بس تسيبه عندك

مثال لفولدر `build/`:
git rm -r --cached build/
بعد ما تعمل `git rm --cached`، لازم:

git add .gitignore
git commit -m "Ignore secret.txt"
