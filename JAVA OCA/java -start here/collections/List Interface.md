الفكرة ببساطة:

- الـ **List** هي **ordered collection** يعني بترتب العناصر بنفس الترتيب اللي انت ضفتهم بيه.
    
- ممكن يكون فيها **duplicate elements** عادي.
    
- كل عنصر بيبقى ليه **index** (من 0 لحد size-1).
    
- هي Interface، يعني مينفعش تعمل منها object مباشرة، لكن فيه كلاسز بتطبقها زي:
    
    - ال`ArrayList`
        
    - ال`LinkedList`
        
    - `الVector` (وأخوها القديم `Stac`
